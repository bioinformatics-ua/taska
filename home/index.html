About TASKA

The system

Taska is an open-source web-based, SaaS proposal for a modular and easily extendable workflow-oriented system designed for managing processes of data extraction and data-handling on multi-user environments where cooperation matters.

This system was developed in the context of EMIF (http://www.emif.eu), a European project that aims to create a common technical and governance framework to facilitate the reuse of health data.

What can you do with TASKA?

The system allows any user to create a study template. A study template is a composition of inter-related tasks, structured in a sequenced way to accomplish specific goals.

After the user select or create a study template, he can run a study. In which users are assigned to tasks with specific deadlines. Each user that runs a study will assume the study manager role.

The task is th basi unit of the system. Everything is a composition of at least one or more tasks.

There are more features assembled to the system, which improve the performance and usability of it. We highlight the following features:

  • Repeatable Workflow Processes via Workflow Templates
  • Simple tasks with I/O and dependencies
  • Form Tasks
  • Processing Tasks
  • Requests management
  • Message system to support the interactions
  • reactJS client interface, with easy-to-use Workflow visual editor
